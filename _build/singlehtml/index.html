SR1000¶

class SR1000.SR1000(interface='USB', ipaddr='192.168.100.100', cmdport=9003, dataport=9004, saveimg=True, timeout=10)¶

Class for Keyence SR1000 bar code reader. The connection is not opened when the object creation.

Parameters:
  • interface – Type of interfacing method, currently only support USB (as we only use USB!)
  • ipaddr – Device IP address
  • cmdport – Command port number
  • dataport – Data port number
  • saveimg – Whether to save the captured image after LON bar code reading
  • timeout – Timeout during the LON bar code reading process
